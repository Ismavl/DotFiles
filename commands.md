# apt
update ubuntu:  
`sudo apt update`

upgrade ubuntu:  
`sudo apt upgrade`

install packages:
`sudo apt install <package-name>`

move directory/file or change its name:
`mv <old> <new>`

# wsl
shut down wsl:
`wsl.exe --shutdown`

start wsl:  
`wsl.exe`

restart ubuntu:
`sudo reboot`

list content of directory:
`ls -al`

create new directory:
`mkdir <name-of-directory>`
